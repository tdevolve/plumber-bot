# Plumber Bot – Project Instructions

## Local dev commands

- Build: npx pnpm build
- Start (dev): NODE_ENV=development npx pnpm start
- Dashboard URL: http://localhost:3000/dashboard

## Git usage

- Run git status before and after major changes.
- Use clear commit messages describing what changed.
- Commit only when tests / basic manual checks pass.

## Assistant behavior

- Always assume I am in a macOS zsh shell.
- Every command block must start with: cd ~/plumber-bot
- Prefer larger multi-step blocks (safe to paste and run).
- Show full file contents with cat > file <<'EOF' for edits.
